# ALMA

Servicios gastronómicos — viandas saludables 100% caseras, sin conservantes, para congelar y hornear.

Proyecto compuesto por tres sub-sistemas, desarrollados en sub-proyectos secuenciales:

1. **Landing + base de backoffice** (listo) — landing pública con contenido editable, más un panel de administración (`/admin`) con login y gestión de contenido/usuarios.
2. **Ecommerce** (próximo) — catálogo, carrito, checkout para clientes minoristas.
3. **Backoffice completo** (futuro) — gestión de pedidos, stock, y el resto de la operación, sobre la base ya armada en `/admin`.

Specs de diseño de cada sub-proyecto en `docs/superpowers/specs/`, planes de implementación en `docs/superpowers/plans/`.

Para levantar el proyecto localmente, crear usuarios admin y deployar, ver [`SETUP.md`](./SETUP.md).
